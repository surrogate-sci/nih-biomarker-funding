# October 2024 Legacy Dataset

This directory contains an early analysis from October 2024 using a different methodology
than the current FY2004-2024 pipeline.

**Key differences from the current pipeline:**
- Data sourced from NIH Reporter **web interface** exports (not bulk ExPORTER CSVs)
- Different keyword strategy and column naming conventions
- Narrower search terms: "biomarker discovery", "surrogate marker", "intermediate endpoint"
- 24,837 projects (vs. 332,324 in the current unified dataset)

**Status:** Kept for reference. Not part of the current analysis workflow.
See `data/filtered/SUMMARY.md` for current dataset statistics.
